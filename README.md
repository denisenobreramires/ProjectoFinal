# 🚀 Portfólio Pessoal — Projeto Final de JavaScript

> Projeto desenvolvido como avaliação final da unidade curricular de JavaScript.

---

## 📋 Descrição

Este projeto é um **portfólio pessoal** desenvolvido com o objetivo de apresentar informações sobre mim — a minha formação, competências, projetos e formas de contacto.

Foi construído de raiz com JavaScript puro, aplicando os conceitos e técnicas aprendidos ao longo da disciplina, como manipulação do DOM, eventos, lógica de programação e interatividade dinâmica.

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
|---|---|
| **HTML5** | Estrutura e semântica das páginas |
| **CSS3** | Estilos, layout e responsividade |
| **JavaScript (Vanilla)** | Lógica, interatividade e manipulação do DOM |

---

## 📁 Estrutura do Projeto

```
portfolio/
├── index.html        # Página principal
├── style.css         # Estilos globais
├── script.js         # Lógica JavaScript
└── README.md         # Documentação do projeto
```

---

## ✨ Funcionalidades

- Apresentação pessoal e percurso académico
- Secção de competências e tecnologias
- Galeria de projetos
- Formulário de contacto
- Navegação dinâmica com JavaScript
